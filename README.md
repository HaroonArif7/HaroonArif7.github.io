## Deploy (GitHub Pages)
1. New public repo: `HaroonArif7.github.io`
2. Upload all files/folders above.
3. Settings → Pages → Source: `main` branch, root.
4. Save. Live URL: `https://HaroonArif7.github.io`

## Edit points
- `index.html`: Bio, skills, contact, Formspree action URL.
- `style.css`: Colors, fonts.
- `projects/` & `blog/`: Add pages as needed.
- `assets/Haroon_Arif_CV.pdf`: Replace with your CV.
